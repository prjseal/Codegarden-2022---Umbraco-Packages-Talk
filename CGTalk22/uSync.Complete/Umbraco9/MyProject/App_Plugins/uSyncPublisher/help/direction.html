A server can be set to accept or decline incoming and outgoing requests from other servers.

Incoming Enabled (default : false)

This server will accept requests from other usync.publisher servers, this means it will allow content to be pushed to or pulled from this server to other configured servers.

Outgoing Enabled (default : true)

This server will show editors the push (and pull) menu items and allow them to push and pull settings to other servers from this server.

appsettings.json

You can enable or disable these settings in your sites appsettings.json file, or other configuration locations depending on how you have setup your hosting environment.

Sample appsettings.json config

"uSync": { 
    "Publisher": {
      "Settings": {
        "IncomingEnabled": true,
        "OutgoingEnabled": true
        }
    }
}
